Requirements:

	1) Node JS version 12.14.1
	2) npm version 6.13.4
	3) PostgreSQL 9.5
	4) pgAdmin 4 version 6.12

========================================================Develop By Ritam Podder=======================================================

To Run the application on your system follow these steps:

	step 1: Create a database name 'crud_operations' in pgAdmin 4.
	step 2: Run my SQL query which is available on 'database.txt' file on 'query tool' of pgAdmin 4.
	step 3: Install node module using command line 'npm i ' in powershell window of your project path.
	step 4: To run the project type 'npx nodemon' on that powershell window.
